# Proyecto de Procesamiento de Imágenes

Este proyecto fue realizado como parte de la materia **Procesamiento de Imágenes** dictada por el docente **Matías Barreto**. 
El objetivo principal es aplicar diferentes técnicas de preprocesamiento, segmentación, modificación de color y análisis 
de imágenes utilizando Python y OpenCV en Google Colab.

## Contenido

El notebook `proc_imagenes1.ipynb` incluye:

- Lectura y visualización de imágenes.
- Reducción de resolución y profundidad de color.
- Segmentación mediante máscaras.
- Detección de contornos y selección de objetos.
- Aplicación de modificaciones de color por zonas.
- Conversión a espacios de color HSV y ajuste de saturación/brillo.
- Comparación visual de los resultados mediante gráficos y concatenaciones.

## Requisitos

- Google Colab o Jupyter Notebook
- Python 3.x
- Librerías:
  - `opencv-python`
  - `numpy`
  - `matplotlib`

## Uso

1. Abrí el archivo `proc_imagenes1.ipynb` en Google Colab.
2. Ejecutá las celdas en orden para seguir el flujo del análisis y transformación de imágenes.
3. Las imágenes modificadas se mostrarán junto a sus versiones originales para una mejor comparación.
