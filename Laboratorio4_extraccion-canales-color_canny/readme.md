# Extracción de Canales de Color y Detección de Bordes con Canny

Este notebook demuestra cómo trabajar con imágenes en Python usando OpenCV y matplotlib. Se enfoca en la extracción de canales de color (RGB) y la aplicación del detector de bordes **Canny** en versiones a color y en escala de grises de una imagen.

## Contenido

- Descarga de una imagen desde la web
- Separación de canales de color (Rojo, Verde y Azul)
- Conversión a formato RGB y escala de grises
- Detección de bordes usando el algoritmo Canny
- Comparación visual entre bordes en color y en gris

## Requisitos

Este notebook utiliza las siguientes bibliotecas:

- `numpy`
- `opencv-python` (cv2)
- `matplotlib`
- `scikit-image` (aunque no se usa directamente en el ejemplo actual)
- `watermark` (opcional)
